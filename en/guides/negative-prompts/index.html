GUIDE_01 / NEGATIVE PROMPTS

What is a negative prompt in music and when should you use one?

A negative prompt is a short list of elements you want to leave out. It does not describe the track you want and it does not replace the main prompt.

IN THIS FILEMain vs negativeWhere it goesVocalsWhen to use itCommon mistakes

Main prompt and negative prompt do different jobs

The main prompt describes the result: genre, pulse, drums, bass, instruments, texture and mix. The negative prompt only names what should stay outside that result.

MAIN PROMPT

Direction, rhythm, low end, sources and finish.

NEGATIVE PROMPT

Elements to exclude, written as a short list.

If the main prompt is weak, a longer exclusion list will not fix it. Start with a clear direction; add exclusions only when they solve a specific problem.

Where does it go?

Paste it into the interface field intended for exclusions. If the tool provides separate fields, keep it out of the main style prompt: mixing “use this” and “do not use this” in one paragraph makes the direction less clear.

Our folders keep both documents separate for exactly that reason: one PDF contains the 30 complete prompts and another contains the 10 optional exclusion lists.

With vocals or without vocals

For an instrumental, use the instrumental setting when one is available. If voices still appear, choose a negative list that explicitly excludes singing, spoken word and vocal phrases. If you want a sung track, do not use that list.

A vocal texture is not always the same as a lead vocal. Wordless grains, chopped breaths or a distant choir can be part of the sound design. Read the purpose of the exclusion before applying it.

When it is useful

  • When vocals appear in an instrumental.
  • When one instrument keeps pulling the result away from the chosen genre.
  • When the finish becomes too bright, clean or cinematic for the folder.
  • When you want to protect a deliberate limit, such as sparse instrumentation.

A generic schematic example would be vocals, bright pop chords, orchestral strings. It is not a second prompt; it is simply three things to leave out.

Three mistakes to avoid

  1. Stacking every list. More exclusions do not mean more control. Pick the one that matches the problem.
  2. Contradicting the main prompt. Do not request a vocal texture and exclude every kind of voice at the same time.
  3. Using it by default. If the first result already stays inside the intended sound, leave the exclusion field empty.
