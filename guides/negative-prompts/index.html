GUIA_01 / NEGATIVE PROMPTS

Qué es un negative prompt en música y cuándo usarlo

Un negative prompt es una lista corta de elementos que quieres dejar fuera. No describe la pista que buscas y no sustituye al prompt principal.

EN ESTE ARCHIVOPrincipal y negativoDónde vaVocesCuándo usarloErrores comunes

El prompt principal y el negative prompt hacen trabajos distintos

El principal describe el resultado: género, pulso, batería, bajo, instrumentos, textura y mezcla. El negative prompt solo nombra lo que debe quedar fuera de ese resultado.

PROMPT PRINCIPAL

Dirección, ritmo, graves, fuentes y acabado.

NEGATIVE PROMPT

Elementos que se excluyen, escritos como una lista corta.

Si la dirección principal es débil, una lista de exclusiones más larga no la arregla. Empieza con una dirección clara y añade una exclusión solo cuando resuelva un problema concreto.

¿Dónde se coloca?

Pégalo en el campo de la interfaz destinado a exclusiones. Si la herramienta ofrece campos separados, no lo añadas al texto principal: mezclar «usa esto» y «no uses esto» en el mismo párrafo hace menos clara la dirección.

Nuestras carpetas mantienen ambos documentos separados por ese motivo: un PDF reúne los 30 prompts completos y otro contiene las 10 listas de exclusión opcionales.

Con voces o sin voces

Para una instrumental, activa el modo instrumental cuando exista. Si aun así aparecen voces, elige una lista que excluya de forma explícita canto, palabra hablada y frases vocales. Si quieres una canción cantada, no uses esa lista.

Una textura vocal no siempre es una voz principal. Granos sin palabras, respiraciones cortadas o un coro lejano pueden formar parte del diseño sonoro. Lee para qué sirve la exclusión antes de aplicarla.

Cuándo resulta útil

  • Cuando aparecen voces en una instrumental.
  • Cuando un instrumento desvía el resultado del género elegido.
  • Cuando el acabado se vuelve demasiado brillante, limpio o cinematográfico para la carpeta.
  • Cuando quieres proteger un límite deliberado, como usar pocos elementos.

Un ejemplo esquemático sería vocals, bright pop chords, orchestral strings. No es un segundo prompt: son únicamente tres elementos que se dejan fuera.

Tres errores que conviene evitar

  1. Apilar todas las listas. Más exclusiones no significan más control. Elige la que corresponde al problema.
  2. Contradecir el prompt principal. No pidas una textura vocal y excluyas a la vez cualquier tipo de voz.
  3. Usarlo por sistema. Si el primer resultado ya se mantiene dentro del sonido buscado, deja vacío el campo de exclusión.

Elige por sonido, no por terminología

Compara las seis carpetas o abre el catálogo. Cada ficha explica el núcleo sonoro, qué cambia entre los 30 prompts y el perfil de su muestra pública.
